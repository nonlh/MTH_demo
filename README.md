# MTH — Market Treasure Hunter

Personal research prototype for collecting and organizing market-related events.

The goal is to turn noisy market information into a structured event database for later analysis.

This is not a trading bot, signal bot, or AI oracle.

## Focus Areas

- Market event collection
- Event normalization
- Asset and ticker detection
- Noise filtering
- Event deduplication
- SQLite storage
- Markdown research reports

## Tech Stack

- Python
- SQLite
- pytest
- JSONL / Markdown

## Status

Local research prototype.

The implementation is private because it contains internal research logic and pipeline details.

Code samples can be shared during an interview if needed.
